## 🔄 Como Funciona

O frontend se comunica com a API automaticamente:

```
Frontend (HTML/CSS/JS)
    ↓
Faz chamadas para /api/*
    ↓
Backend (Express)
    ↓
Retorna JSON
    ↓
Frontend exibe na tela
```

**Tudo serverless no Vercel!** 🚀
